## 🔥 工具详情

### 1️⃣ 选文字数统计（油猴脚本）🎯

> **一句话：选中网页上的文字 → 右键 → 右上角弹出字数统计面板**

✅ 统计总字符（含空格 / 不含空格）、汉字数、英文单词数、数字串、标点符号、行数
✅ 固定右上角显示，**不会被右键菜单遮挡**
✅ mousedown 捕获阶段缓存选区，**即使页面取消选中也能正常统计**
✅ 带关闭按钮 + ESC 快捷键关闭

**安装方式：**
1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开 [`选文字数统计.user.js`](选文字数统计.user.js)
3. Tampermonkey 自动识别并提示安装

---

### 2️⃣ Memo 备忘录 🗒️

> **像贴便利贴一样在页面上自由摆放你的想法和图片！**

✅ 双击空白处创建新卡片
✅ 自由拖拽排列位置，支持整理/散开模式
✅ 卡片之间可以建立关联连线
✅ 支持粘贴图片、拖入文件（PDF/Word/Excel 等）
✅ 全文搜索功能
✅ 一键导出为文本

**使用方式：** 直接浏览器打开 [`Memo.html`](Memo.html) 即可，数据自动保存在本地浏览器中

---

### 3️⃣ 行测刷题计时器 ⏱️

> **公考行测刷题人的秘密武器！倒计时 + 正计时二合一**

🔹 **主计时器（倒计时）：** 设置秒数 → 开始 → 最后5秒变红提醒 + 每秒滴答声 → 到0后双蜂鸣声并自动重新开始
🔹 **正计时秒表（右下角）：** 可折叠的秒表功能，随时记录做题总时长
🔹 支持空格键快捷操作，手机端完美适配

**使用方式：** 直接浏览器打开 [`行测计时器.html`](行测计时器.html)

---

### 4️⃣ 记忆闪卡 🃏

> **用艾宾浩斯遗忘曲线的科学方法来背东西！翻转卡片让记忆更牢固**

🃏 点击卡片翻转查看答案（3D 翻转动画）
📥 支持批量导入（`正面 | 背面` 格式）
📊 进度追踪：待学习 / 已掌握 分库管理
🔀 打乱顺序、编辑单张卡片
🌙 支持暗色模式（跟随系统）
💾 导出/导入 JSON 备份
⌨️ 键盘快捷键支持（←→翻页、空格翻转）

**使用方式：** 直接浏览器打开 [`记忆闪卡.html`](记忆闪卡.html)

## 💡 为什么这些工具值得用？

- **零依赖！零安装！** 所有工具都是单个 HTML 文件或 JS 文件，不需要 Node.js / Python / 任何后端
- **数据安全** — 数据全部存在你自己的浏览器 localStorage 里，不会上传到任何服务器
- **离线可用** — 断网也能正常使用
- **开源免费** — MIT 协议，随便改随便用
